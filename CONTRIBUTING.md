# 기여 가이드

AI Visibility Optimizer 프로젝트에 기여해주셔서 감사합니다!

## 기여 방법

1. **Fork** 저장소
2. **Feature 브랜치** 생성 (`git checkout -b feature/AmazingFeature`)
3. **Commit** 변경사항 (`git commit -m 'Add some AmazingFeature'`)
4. **Push** 브랜치 (`git push origin feature/AmazingFeature`)
5. **Pull Request** 생성

## 코딩 스타일

- ES6+ 문법 사용
- async/await 사용
- 에러 처리 필수
- 주석 작성 권장
